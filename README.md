# RetroMarket
중고 거래 판매 애플리케이션 (Spring X Vue)

### 페이지 소개

#### master 브렌치
> 기술 스택 및 프로젝트 간략히 소개

<br>

#### page-introduce 브렌치
> 페이지 및 소개

<br>

#### main-page-ui 브렌치
> 메인 페이지 UI

<br>

#### main-page-ui-v2 브렌치
> 기존 main-pageui 브렌치 -> v2.0 업데이트

<br>

#### main-page-gsap 브렌치
> main-page gsap 애니메이션 연동 및 구현

<br>

#### main-product-paging-source 브렌치
> 중고 거래에서 페이징 처리 구현

<br>

#### main-product-filtering-source 브렌치
> 중고 거래에서 주소 필터링 처리 구현

<br>

#### source-kakao-map-api 브렌치
> 카카오 지도 API 관련

<br>
